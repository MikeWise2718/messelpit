# Messel Pit — HVBG tile manifest

# Geodaten-online shop (see tools/download_messel_data.py for URLs) and
# unzip into data/raw/dgm1/ and data/raw/dop20/. prep_rasters.py derives
# the bbox from whatever DGM1 tiles it finds, so the exact list here
# defines the coverage.
#
# Naming: <dataset>_32_<E_km>_<N_km>_1_he.<ext>
#   32   = UTM zone 32N (EPSG:25832)
#   E_km = SW corner easting in km
#   N_km = SW corner northing in km
#   1    = tile edge length in km
#   he   = Hessen
#
# Coverage: 29 tiles, irregular footprint roughly
#   480 000 .. 486 000 E  ×  5 526 000 .. 5 535 000 N
